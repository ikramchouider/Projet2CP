vous trouvez une visualisation qui explique comment utiliser le site pour simuler vos programmes et instructions . Aussi avec une description ecrite et bien detaillée.

Get started

Suivez Les étapes suivantes

étape :

Allez vers l'éspace de la simulation ( dans la page Acceuil).

étape 2 :

Clickez sur " fichiers " pour choisir le programme que vous voulez simulez .

étape 3 :

Le programme choisis va étre écrit dans l'éspace du texte .

Vous pouvez aussi écrire votre propre instructions , en respectant la syntaxe .

étape 4 :

Clickez sur "Assembler" pour détecter les erreurs de syntaxe.

Vous pouvez ensuite faire "Exécuter" pour voir le résultat de votre programme.

étape 5 :

Clickez sur "Simuler" pour voir la simulation graphique du votre programme sur l'architecture de la machine.
