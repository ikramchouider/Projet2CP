Apprenez à utliser PASSE

Vous trouvrez une visualisation qui explique comment utiliser
le site pour simuler vos programmes et instructions . Aussi,
vous trouvrez une description sur les differentes étapes pour
se familiariser avec le site, écrites et bien detaillée.



Get started

Suivez Les étapes ci-dessous

Etape 1 :

Allez vers l'éspace de simulation ( dans la page d'Acceuil ).

Etape 2 :

Clickez sur " fichiers " pour choisir le programme que vous voulez simuler.

Etape 3 :

Le programme choisi apparaîtra dans la partie réservée au code assembleur ainsi que sa traduction dans la zone du code héxadécimal. Vous pouvez aussi écrire votre propre programme, en respectant la syntaxe.

Etape 4 :

Clickez sur "Assembler" pour détecter les erreurs de syntaxe. Vous pouvez clicker sur "Traduire" pour voir le code équivalents en héxadécimal ou en mnémaunique.

Etape 5 :

Clicker sur "Exécuter" pour voir le résultat de votre programme.

Etape 6 :

Sur la gauche de cette console, vous trouverez deux button : celui en haut permet d'importer des fichiers d'après votre machine (.PASSE ou .txt), et le deuxième pour sauvegarder le code dans votre machine dans un fichier .PASSE .

Etape 7 :

Clickez sur "Simuler" pour voir la simulation graphique de votre programme sur l'architecture de la machine.
